===========================================
Felo – estimate the fencing strength
===========================================

:Date: $Date$
:Author: Torsten Bronger <bronger@physik.rwth-aachen.de>


Summary
============

Felo ratings are a wonderful new method to estimate fencers.  The Felo program
calculates these ratings for a given group of fencers.  The only thing
necessary is to provide the program with a bout result list.  The program
offers a graphical user interface.
